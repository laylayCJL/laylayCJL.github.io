Java流程控制语句

一、顺序结构

二、选择结构

1、if语句注意事项

  • 关系表达式结果必须为boolean类型
  • if控制多条语句时要有大括号
  • if的条件后面很容易不小心加上了分号

2、if语句与三元运算符的转换

注意：当if语句控制的语句体（表达式a/表达式b）是一条输出语句的时候，就不成立，因为三元运算符是一个运算符，必须要有一个结果返回，而输出语句无法作为一个结果被返回

3、switch语句

（1）格式的解释：switch语句表达式的取值可以是

  • byte，short，int，char
  • JDK5以后可以是枚举
  • JDK8以后可以是字符串string

（2）switch语句的注意事项

  • case后只能跟常量，且case值不能重复
  • 结束标志：break或者是整个switch执行完毕

三、循环结构

1、for循环

适用于明确知道循环次数

先判断执行条件再执行循环体

for循环定义的控制变量仅在for循环内有效，跳出for循环之后就失效，即从内存中消失

2、while循环

适用于不明确循环次数

先判断执行条件再执行循环体

3、do-while循环

至少执行一次循环体

4、控制跳转语句

（1）goto语句
  • java保留字 不建议使用
（2）break中断

  • 在选择switch和循环里用，其他环境用没有意义

  • 作用：①调出单层循环 (更常用)

    ​ ②要想用break跳出多层循环，就要吧你想跳出的循环起个名字，然后break 名字；

（3）continue继续
  • 只在循环中使用，其他环境没有意义
  • 作用：退出本次循环，继续下一次循环的执行
  • 与break的区别：break会退出整个一层循环，而continue只退出本次循环
（4）return返回
  • return关键字不是为了跳出循环体，更常用的功能是结束一个方法，跳转到上层调用的方法
